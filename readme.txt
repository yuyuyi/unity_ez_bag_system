1. You can open the scene through the unity file under Assets / Scenes.

2. All script source code is stored in Assets / script file.

PlayerMovementb: Mount on the character, control the character movement. Players can press "B
"Open the backpack.
itemMain: Mounted on the item, used to control the item and the character to disappear after collision and count into the backpack.
IventoryManager: Display items in the backpack.
slot: Display items related.

3. All the resource files used are in Assets / resoure file, find it by yourself.

4. Display the picture of the item + the prefab of the character in the inventory, in Assets / Prefab.

5. In the inventoryscript folder, the source code of object objects and backpack library objects is stored. Inheriting ScriptableObject allows us to use the right mouse button to create objects directly in unity.

6. The items created with the right mouse button are stored in the items.

1.アセット/シーンの下のユニティファイルからシーンを開くことができます。

2.すべてのスクリプトソースコードはAssets /スクリプトファイルに保存されます。

PlayerMovementb：キャラクターにマウントし、キャラクターの動きを制御します。 プレイヤーは「B
「バックパックを開けて。
itemMain：アイテムにマウントされ、アイテムとキャラクターが衝突後に消えるように制御するために使用され、バックパックでカウントされます。
IventoryManager：バックパックのアイテムを表示します。
スロット：関連するアイテムを表示します。

3.使用されるすべてのリソースファイルはAssets / resoureファイルにあります。自分で見つけてください。

4.アイテムの写真とキャラクターのプレハブを、アセット/プレハブのインベントリに表示します。

5. inventoryscriptフォルダーには、オブジェクトオブジェクトとバックパックライブラリオブジェクトのソースコードが格納されています。 ScriptableObjectを継承すると、マウスの右ボタンを使用して、オブジェクトを1つに直接作成できます。

6.マウスの右ボタンで作成したアイテムがアイテムに保存されます。
